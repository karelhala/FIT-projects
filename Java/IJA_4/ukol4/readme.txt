Odevzdani milestonu. Projekt umi nacist data z XML souboru a zobrazit petriho site. Je mozne editovsat jednotlive uzly. Dale je mozne pohybovat s jednotlivimy uzlu a pridavat spojeni. Nebylo jeste implementovano odstranovani jednotlivych uzlu.

Aplikace obsahuje 3 zaklikavaci tlacitka, 3 klasdicke tlaciteka a jedno menu.
Add place prida misto pro uchovani promennych, zakliknout tlacitko a pote kikat na pracovni plochu.
Add Trnasmition prida misto pro uchovani vyrazu, zakliknout tlacitko a pote kikat na pracovni plochu.
Add joint. Zakliknout tlacitko a muzete pridavat prechody. Kliknete na prvni objekt a pote kliknete na druhy. Netahnout z objektu k objektu, pouze kliknout na prvni a pote na druhy. A vygeneruje se sipka od jednoho objektu k druhemu.
Je mozne presouvat jednotlive mista. Pouze je nutne nemit zakliknute "add joint".
Tlacitkem Save lze ulozit stavajici petriho sit.
Tlacitkem load lze otevrit vygenerovane XML.
Tlacitkem clear all lze vycistit kompletne plochu.
Menu je stae nefunkcni.
